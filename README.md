# 简易记账

## 本地环境

- 操作系统: Windows 10 版本 ( 10.0.19042 ) 
- 浏览器: Google Chrome 
最新版本 91.0.4472.124 ( >=86 )
- Nodejs: 14.16.0

## 项目准备

```bash
$ git clone https://github.com/QAQmmttyyy/mini-account-book.git
$ cd mini-account-book
$ npm install
```

## 开发运行

```bash
$ npm run dev
```

## 测试

```bash
$ npm run test
```
